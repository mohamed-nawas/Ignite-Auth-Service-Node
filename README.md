## Ignite Tuition

This is the node version of the authorization microservice for the Ignite project backend.

## Requirements

- Node
- Typescript

## Dependencies

All dependencies are available in package.json

## Build

```
tsc
```

## Run

```
npm install
npm run start or npm run dev
```

or

```
tsc
node ./dist/index.js
```

## Test

```
npm run test
```

## Postman Collection

src/utils/postman_collection.json
